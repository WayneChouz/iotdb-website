# JMX工具

Java VisualVM提供了一个可视化的界面，用于查看Java应用程序在Java虚拟机（JVM）上运行的详细信息，并对这些应用程序进行故障排除和分析。

# 使用

第一步：启动IoTDB server。

第二步：建立连接。对于本地的监控，不需要手动配置连接。对于远程的监控，可以添加远程的ip地址，并将端口配置为31999。

第三步：开始监控。双击你所建立的ip地址，就可以看到执行应用的详细信息。
